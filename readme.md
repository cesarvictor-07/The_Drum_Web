## The Drum Web
Welcome to The Drum Web! (TDW for short) A web made for people who are unaware of what creates a drum-kit, all of its parts and some extras that you can check out in this web.

---

## Description
As of right now the web shows the minimum that makes a drum-kit, the drums, the cymbals and an extra section, later on the web will evolve and show even more pieces and add more fucntions like full on descriptions and add the sound of each piece, you will also be able to create an account on the website just in case if you want to be updated with the web's content
